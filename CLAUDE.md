# AI Assistant Instructions

When working in this repository, please observe the custom workflows defined in the `.agents/workflows/` directory. 

- These `.md` files contain explicit step-by-step instructions for specific commands.
- If the user provides a command that matches a filename in `.agents/workflows/` (e.g., "save atp" for `save_atp.md`, or "commit" for `commit.md`), you MUST read that file and follow its instructions exactly as written.
